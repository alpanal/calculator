# calculator
 # 🔢 Calculator App

An interactive **Calculator Web Application** built using HTML, CSS and JavaScript.  
This project performs basic arithmetic operations with a clean, responsive and user-friendly interface.

## 🚀 Features

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)
- Clear Button (C)
- Delete Button (⌫)
- Real-time Display
- Responsive Design

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript

```bash
calculator-app/

```

## ⚙️ Installation

Clone repository:

```bash
git clone https://github.com/your-username/calculator-app.git
```

Move to project folder:

```bash
cd calculator-app
```

Run project:

```bash
Open index.html in browser
```

Or use VS Code Live Server.

## ✨ How It Works

- User clicks number buttons for input  
- Operator buttons perform calculations  
- JavaScript handles expression logic  
- Display updates in real time
